# AI Personal Memory & Decision Assistant

A production-level AI-powered personal memory system that acts as a secure "second brain" for users.

The application allows users to store, organize, search, and interact with their personal information using Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), vector embeddings, and knowledge graphs.
